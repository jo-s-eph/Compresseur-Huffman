Compiler le code : 
make 

executer le code : 
./main.o

Le code se divise en plusieurs parties :

- huffman : Contient une structure de tableau ainsi que les fonctions d'encodage et de decodage de huffman
- text : Contient les fonctions necessaires pour traiter du texte (lire, encoder, decoder) ainsi que des fonctions annexes
- tree : Contient la structure d'un arbre binaire modifié avec ses fonctions associées
- tas : Contient la structure d'un TAS min. modifié avec ses fonctions associées

Le fichier main apelle l'ensemble necessaire afin d'encoder puis de decoder en fonction du fichier demandé.

Une fois executé, suivez les indications du menu et passez toujours des fichiers existants.
